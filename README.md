# Symfony Docker
A adchieve dev tools project using docker with the [Symfony](https://symfony.com) web framework,
with [FrankenPHP](https://frankenphp.dev) and [Caddy](https://caddyserver.com/) inside!

## Getting Started
1. If not already done, [install Docker Compose](https://docs.docker.com/compose/install/) (v2.10+)
2. Run `docker compose build --no-cache` to build fresh images
3. Run `docker compose up --pull always -d --wait` to set up and start a fresh Symfony project
4. Open `https://atools.localhost` in your favorite web browser and [accept the auto-generated TLS certificate](https://stackoverflow.com/a/15076602/1352334).
5. Configure host system to trust the Certificate Authority used in the Caddy container, see [TLS Certificates](docs/tls.md).
6. Run `docker compose down --remove-orphans` to stop the Docker containers.
